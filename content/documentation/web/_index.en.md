## What is the Trigo Theme?

Trigo is a simple, modern, fast, feature-packed and flexible [Hugo][hugo] theme to build static websites for documentation, blogs, portfolios and many more.

Hugo is a one of the most popular static site generator. What sets Hugo apart is that it is a single binary, making it easy to install and run on various platforms. It is also extremely fast and reliable, capable of rendering a site with thousands of pages in milliseconds.

Trigo is custom made and built with a mindset focused on having a minimal footprint. To get started, no extra dependencies are required; all you need is a single YAML configuration file, along with your Markdown content. Focus is on writing quality content instead of setting up tooling.

## Features

- **Responsive Layout and Dark Mode** - It looks great on all devices, from mobile, tablet to desktop. Dark mode is also supported to accommodate various lighting conditions.
- **Fast and Lightweight** - Powered by Hugo, a lightning-fast static-site generator housed in a single binary file, its footprint is minimal. No JavaScript are needed to use it. Additional JavaScript features are optional and can be disabled easily.
- **Full-text Search** - Built-in offline full-text search powered by FlexSearch, no additional configuration required.
- **Features** - Markdown, syntax highlighting, LaTeX math formulae, diagrams and Shortcodes elements to enhance your content. Table of contents, breadcrumbs, pagination, sidebar navigation and more are all automatically generated.
- **Multi-language and SEO Ready** - Multi-language sites made easy with Hugo's multilingual mode. Out-of-the-box support is included for SEO tags, Open Graph, sharing to social media and many more.
