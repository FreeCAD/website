This week in FreeCAD development:

**PartDesign**: wwmayer and FlachyJoe fixed 4 issues including two release blockers.

**Part**: wwmayer fixed the PropertyGeometryList::Save output to save XML attributes into one line.

**Assembly**: PaddleSttoke and furgo16 fixed 12 bugs including 3 release blockers and 2 crashers.

**FEM**: marioalexis84 fixed several bugs including one release blocker. One of the fixes touches the vertical size of spinboxes in the task panel - this seems to be a "popular" bug recently.

**BIM**: adrianinsaval, yorikvanhavre, Roy-043, and paullee0 fixed several bugs including 1 release blocker.

**TechDraw**: WandererFan and Syres916 fixed several issues including one release blocker.

**CAM**: ronaksingh27 fixed the Marlin post-processor code, and shaise fixed shader code compilation on Linux.

**GUI**: benj5378, pinkavaj, pieterhijma, and Syres916 fixed a couple of crashers and one release blocker.

Among other changes:

- wwmayer fixed an issue in Sketcher and two issues in the core.

- More fixes all around from kriswilk, hyarion, and MisterMakerNL.

**PR stats**: since the previous report, 47 pull requests have been merged, 25 new pull requests have been opened.

**Issue stats**: overall, there are 2182 open issues in the tracker, up by 25 from last week. 15 of them are v1.0 release blockers, down by 5 from last week.
