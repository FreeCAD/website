The FreeCAD team is happy to announce the release of [FreeCAD 0.21.2](https://github.com/FreeCAD/FreeCAD/releases/tag/0.21.2)! This is a bug fix release for the 0.21 release cycle. A detailed list of back ported fixes can be found [on GitHub](https://github.com/FreeCAD/FreeCAD/compare/0.21.1...0.21.2).

Get the latest version at [our downloads page](https://www.freecad.org/downloads.php)!
